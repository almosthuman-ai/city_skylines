# Run and deploy your app

This contains everything you need to run your app locally and deploy it to GitHub Pages.

## Local + Deploy (GitHub Pages)

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app locally:
   `npm run dev`
3. Build for GitHub Pages (outputs to `/docs`):
   `npm run build`
4. In GitHub: Settings → Pages → Deploy from a branch → `main` / `/docs`
